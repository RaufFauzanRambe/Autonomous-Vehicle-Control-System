# Research Papers

This directory contains research papers and references relevant to the Autonomous Vehicle Control System.

## Paper Categories

| Category | Topics |
|----------|--------|
| Perception | Object detection, tracking, sensor fusion |
| Localization | SLAM, state estimation, map matching |
| Planning | Path planning, behavior planning, trajectory optimization |
| Control | MPC, PID, lateral control, vehicle dynamics |
| V2X | DSRC, C-V2X, cooperative driving |
| Safety | Functional safety, fault tolerance, verification |

## Key Papers

See [docs/research_notes.md](../research_notes.md) for detailed summaries and design decisions informed by these papers.

## Adding Papers

1. Place PDF files in this directory with naming convention: `AuthorYear_ShortTitle.pdf`
2. Add citation information to the references section in `research_notes.md`
3. Summarize key findings and relevance to AVCS in the research notes

## Citation Format

```
Author(s). (Year). "Title." Conference/Journal. [URL or DOI]
```
